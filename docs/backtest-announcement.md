# Backtest Feature Announcement

One announcement serves both channels. The admin form sends the same title and
message to the in-app notification bell (plain text) and the Telegram channel
(title auto-bolded). So the message below uses no Markdown syntax, only line
breaks and emojis, which render correctly in both.

**Title** (max 100 chars):

🧪 New: Backtest your buying rules

**Message:**

Does your dip-buying strategy actually beat plain DCA? Stop guessing, start measuring.

You can now run any buying rule set against real historical prices and compare it side by side with a plain DCA baseline: final value, ROI, average buy price, and every boosted buy marked on the chart.

Try it: open Rule sets, click the flask icon 🧪 on any buying rule set, pick an asset and a date range, and run. Also available on your plans.

Works with all major crypto assets like BTC, ETH and SOL. Past performance does not predict future results, but at least now you will know your history. 📈
